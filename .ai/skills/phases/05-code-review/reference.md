## Finding template

```markdown
### SEC-001: Validate callback destination

- **Severity:** blocking
- **Evidence:** `src/notifications/email.ts:41`
- **AC/design reference:** AC-001; api-design.md §Callback
- **Observation:** Untrusted callback URLs reach the HTTP client without an allowlist.
- **Impact:** A caller can request internal network resources.
- **Required action:** Validate scheme and host against the configured allowlist.
```

For a non-blocking finding, replace **Required action** with **Suggestion**. If a line number is unavailable, cite the repository path and the exact deletion diff, generated output, or configuration section inspected.
